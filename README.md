# OOF!

Dalamud plugin that plays the oof sound when you take 
- fall damage
- die
- party/alliance members die
- run /oof command

#### whimiscal features!
 - use a custom sound file!
 - instant access to the hour long documentary on oof by hbomberguy!
 - fully featured settings window!
### play oof sound:
```
/oof
```
### change settings:
```
/oofsettings
```
